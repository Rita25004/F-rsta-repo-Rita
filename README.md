Här kommer några förklaring av ord och begrepp inom programering:
# Vad är Git:
Git är ett program, där kan man spara olika versioner av ett projekt och gå tillbaka om något går fel.

Det finns några begrepper inom Git som man behöver förstå:
### Repository (repo)
där förändringshistorik lagras
### Commit
betyder att en sparad version av projekt
### Branch (gren)
En branch används för att arbeta med olika utvecklingsspår parallellt. Att skapa nya versioner utan att förstöra huvudversionen.
### Merge
Betyder att slår ihåp en branch med en annan.
### Checkout
betyder att att byta till en annan version eller gren av projektet.
### Push
betyder att skicka commits (ändringar) från dator till t.ex. Github.
### Pull
Hämtar ändringar från t.ex. Github till dator.
### Clone
betyder att kopierar ett helt Git-repo t.ex. från Github till dator.
